### Multi-Agent Deep Research Assistant
An AI system where multiple agents collaborate to research a topic, verify sources
and generate detailed reports automatically.

Research can be time-consuming—searching across multiple sources, filtering information, and organizing insights. 
I wanted to build a system that makes this process faster and more structured.

##🔹 What it does

🌐 Searches the web for relevant information
🤖 Uses multiple AI agents for different research tasks
📑 Generates structured research reports
🔍 Provides citations for better transparency


## Features

* Web research
* Fact verification
* Source collection
* Report generation
* Presentation creation


## Tech Stack
Python, LangGraph, OpenAI GPT-5, Tavily, FastAPI, PostgreSQL
